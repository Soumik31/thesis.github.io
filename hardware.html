Operating Principles

In our hardware installation we used vehicle counter for counting number of vehicles on the road on a particular time. Inside of vehicle counter we have Arduino Nano, Pressure sensor (MPXV5050DP), LCD Display, GSM Module (SIM808) and LED. Outside of vehicle counter we have rubber tube and power bank. Using the road tube and pressure sensor we took data from road and then instantly we send that data to server using GSM module and the data will be on server and then matlab will download data it will do kalman filtering and then it will again send back to server. Then Android app will take data and do some calculations and then the traffic density will show on app in the map for the next weekâ€™s same day and time.
