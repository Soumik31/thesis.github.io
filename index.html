Things to know

Day by day traffic jam is becoming a very big problem for all of us. Every problem finds a way to its solution same for traffic jam we used a filter named “Kalman Filter”. This filter will take data from road using vehicle counter and it will predict vehicle density for next week’s both same day and same time on the same road from where data has been taken. This filter is named after Hungarian born American Electrical and Electronics Engineer Rudolf E. Kalman after he invented Kalman Filter which was first used in heat seeking missiles and then he was rewarded in October 7, 2009 from Barack Obama because values come from prediction of Kalman Filter has an error of 0.01% and previously no filter had so much less error as Kalman Filter. Now we are using this filter to predict traffic on road and people will see this predicted value and they can take decision before sitting on driver’s seat.
